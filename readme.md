## Action Tags

Add tags to document title and folder title (ex. `Document Title #i #70%`)

- GTD
	- `#inbox` or `#i`
	- `#snoozed` or `#s`
- Mark document to unimportant
	- `#-1`
- Progress
	- `#0%` ~ `#100%`
